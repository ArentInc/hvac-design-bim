### TC-001: 冷水流量計算（標準入力）

- **テスト名**: 冷水流量計算 -- AHU冷房能力50kWから流量を正しく算出する
  - **何をテストするか**: coolingCapacity から冷水流量（m3/s）を算出する計算式の正確性
  - **期待される動作**: `flowRate = coolingCapacity / (cp * deltaT * rho)` の公式に従い流量が算出される
- **入力値**: `coolingCapacity = 50 kW`, `deltaT = 5 K`, `cp = 4.186 kJ/(kg*K)`, `rho = 1000 kg/m3`
  - **入力データの意味**: 中規模 AHU の一般的な冷房能力値。PRDセクション15.7で使用されている値
- **期待される結果**: `flowRate ≈ 0.002389 m3/s`（= 50 / (4.186 * 5 * 1000)）、約 8.6 m3/h
  - **期待結果の理由**: 物理的公式に基づく計算結果。TASK-0036 テスト1の期待値と一致
- **テストの目的**: 冷水流量計算の基本精度確認
  - **確認ポイント**: 相対誤差 0.1% 以内で一致すること
- 🔵 TASK-0036 テスト1 に明記された Given/When/Then と完全一致

  - **何をテストするか**: `diameter = sqrt(4 * flowRate / (pi * velocity))` の計算式

- **期待される結果**: 計算口径 ≈ 45mm（`sqrt(4 * 0.00239 / (pi * 1.5)) * 1000 ≈ 45.0mm`）
  - **期待結果の理由**: 平方根公式による算出。TASK-0036 テスト2の期待値と一致
- **テストの目的**: 理論口径算出の精度確認
  - **確認ポイント**: mm 単位で整数近似が妥当であること
- 🔵 TASK-0036 テスト2 に明記

### TC-006: 等価長さ法による圧損概算

- **テスト名**: 等価長さ法圧損概算 -- 直管長20m・50A口径で物理的に妥当な圧損を算出
  - **何をテストするか**: ダルシー・ワイスバッハ式と等価長さ法による圧損概算計算
  - **期待される動作**: `ΔP = lambda * (totalLength / d) * (rho * v^2 / 2)` で圧損が算出される
- **入力値**: 直管長 = 20m, nominalSize = 50A (innerDiameter = 0.0527m), flowRate = 0.00239 m3/s, lambda = 0.02, fittingFactor = 0.5
  - **入力データの意味**: 一般的な配管区間長。TASK-0036 テスト7 に明記された入力値
- **期待される結果**:
  - 総等価長さ = 20 * (1 + 0.5) = 30m
  - 実流速 v ≈ 1.10 m/s
  - ΔP = 0.02 * (30 / 0.0527) * (1000 * 1.10^2 / 2) ≈ 6.9 kPa
  - 物理的に妥当な範囲（数 kPa ~ 十数 kPa）
  - **期待結果の理由**: 鋼管概算値と標準パラメータによる計算。数 kPa が妥当
- **テストの目的**: 圧損概算計算の物理的妥当性確認
  - **確認ポイント**: 0 kPa より大きく、かつ 100 kPa 未満の妥当な値であること
- 🔵 TASK-0036 テスト7 に明記
