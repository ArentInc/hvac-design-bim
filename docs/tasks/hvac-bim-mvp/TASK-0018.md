### 1. 🔵 ZoneDrawToolでの面積リアルタイムオーバーレイ (REQ-1601)

TASK-0013のZoneDrawToolを拡張し、描画中のポリゴン面積をリアルタイムで3D空間内にオーバーレイ表示する。

**ファイル**: `packages/editor/src/components/tools/hvac/zone-draw-tool.tsx` （TASK-0013で作成済みのファイルを拡張）

```typescript
// ポリゴン中心に面積テキストを表示
function AreaOverlay({
  vertices,
  currentPoint,
}: {
  vertices: { x: number; y: number }[]
  currentPoint: { x: number; y: number } | null
}) {
  const allPoints = currentPoint ? [...vertices, currentPoint] : vertices

  if (allPoints.length < 3) return null

  const area = calculatePolygonArea(allPoints)
  const center = calculatePolygonCenter(allPoints)

  return (
    <Html
      position={[center.x, floorHeight + 0.5, center.y]}
      center
      style={{ pointerEvents: 'none' }}
    >
      <div className="area-overlay">
        <span className="area-value">{area.toFixed(1)}</span>
        <span className="area-unit">m²</span>
      </div>
    </Html>
  )
}

function calculatePolygonCenter(
  vertices: { x: number; y: number }[]
): { x: number; y: number } {
  const sumX = vertices.reduce((sum, v) => sum + v.x, 0)
  const sumY = vertices.reduce((sum, v) => sum + v.y, 0)
  return {
    x: sumX / vertices.length,
    y: sumY / vertices.length,
  }
}
```

- R3Fの`<Html>`コンポーネントで3D空間内にHTML要素をオーバーレイ
- マウス位置（currentPoint）を含む仮ポリゴンの面積を表示
- ポリゴン重心位置に配置
- pointerEvents: 'none'でインタラクションを阻害しない
- 2頂点以下の場合は非表示

**面積表示スタイル**:

### 2. 🔵 HvacZoneRendererでのcalcResult検出→カラーフェードイン (REQ-1602)

TASK-0012のHvacZoneRendererを拡張し、負荷計算完了時にカラーフェードインアニメーションを実装する。

**ファイル**: `packages/viewer/src/components/renderers/hvac/hvac-zone-renderer.tsx` （TASK-0012で作成済みのファイルを拡張）

```typescript
const ZONE_DEFAULT_COLOR = '#9E9E9E' // グレー（calcResult未設定時）

export function HvacZoneRenderer({ nodeId }: { nodeId: string }) {
  const node = useScene((s) => s.nodes[nodeId]) as HvacZoneNode
  const materialRef = useRef<MeshBasicMaterial>(null)
  const [prevCalcResult, setPrevCalcResult] = useState<boolean>(false)
  const fadeProgress = useRef(0)

  const targetColor = useMemo(() => {
    if (!node.calcResult) return ZONE_DEFAULT_COLOR
    return ZONE_USAGE_COLORS[node.usage] ?? ZONE_DEFAULT_COLOR
  }, [node.calcResult, node.usage])

  // calcResultが新たに設定された検出
  useEffect(() => {
    if (node.calcResult && !prevCalcResult) {
      fadeProgress.current = 0 // フェードイン開始
    }
    setPrevCalcResult(!!node.calcResult)
  }, [node.calcResult, prevCalcResult])

  // useFrameでアニメーション更新
  useFrame((_, delta) => {
    if (!materialRef.current) return
    if (fadeProgress.current >= 1) return

    fadeProgress.current = Math.min(fadeProgress.current + delta * 2, 1) // 0.5秒間

    const fromColor = new Color(ZONE_DEFAULT_COLOR)
    const toColor = new Color(targetColor)
    const currentColor = fromColor.lerp(toColor, fadeProgress.current)

    materialRef.current.color.copy(currentColor)
  })

  return (
    <mesh ref={ref} layers={ZONE_LAYER}>
      <shapeGeometry args={[shape]} />
      <meshBasicMaterial
        ref={materialRef}
        color={targetColor}
        transparent
        opacity={0.4}
        side={DoubleSide}
      />
    </mesh>
  )
}
```

- calcResult未設定 → グレー(#9E9E9E)
- calcResult設定検出 → グレーから用途別カラーへ0.5秒フェードイン
- Color.lerp()で滑らかな色遷移
- useFrameでフレームごとにアニメーション更新

### 3. 🟡 フェードインアニメーション実装（Three.js material opacity transition）

フェードインの詳細実装。

```typescript
// アニメーション設定
const FADE_DURATION = 0.5 // 秒
const FADE_SPEED = 1 / FADE_DURATION // 2.0

// イージング関数（ease-out）
function easeOut(t: number): number {
  return 1 - (1 - t) ** 2
}

// useFrame内でのアニメーション
useFrame((_, delta) => {
  if (!materialRef.current) return
  if (fadeProgress.current >= 1) return

  fadeProgress.current = Math.min(
    fadeProgress.current + delta * FADE_SPEED,
    1
  )

  const easedProgress = easeOut(fadeProgress.current)
  const fromColor = new Color(ZONE_DEFAULT_COLOR)
  const toColor = new Color(targetColor)
  fromColor.lerp(toColor, easedProgress)

  materialRef.current.color.copy(fromColor)
})
```

- ease-outイージングで自然なフェードイン
- deltaTimeベースでフレームレート非依存
- アニメーション完了後はuseFrameの計算をスキップ（パフォーマンス）
