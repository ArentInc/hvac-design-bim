## 実装詳細

### 1. 🔵 SystemPanel — 系統プロパティパネル

`packages/editor/src/components/panels/hvac/system-panel.tsx` に系統プロパティパネルを作成する。

```typescript
// packages/editor/src/components/panels/hvac/system-panel.tsx
import { useScene } from '@pascal-app/core'

interface SystemPanelProps {
  nodeId: string
}

export function SystemPanel({ nodeId }: SystemPanelProps) {
  const node = useScene(state => state.nodes[nodeId])
  if (!node || node.type !== 'system') return null

  return (
    <div>
      <h3>系統プロパティ</h3>
      {/* systemName */}
      {/* servedZoneIds一覧 */}
      {/* aggregatedLoad (冷房/暖房/風量) */}
      {/* status */}
      {/* 紐付きAHU情報 */}
    </div>
  )
}
```

表示項目:
- **systemName**: 系統名称（編集可能）
- **servedZoneIds一覧**: 所属ゾーン名のリスト表示
- **aggregatedLoad**: 合算冷房負荷、合算暖房負荷、合算必要風量
- **status**: 系統ステータス（正常/警告/エラー）
- **AHU情報**: 紐付きAHUのequipmentName、定格値サマリー

### 2. 🔵 AhuPanel — AHUプロパティパネル

`packages/editor/src/components/panels/hvac/ahu-panel.tsx` にAHUプロパティパネルを作成する。

```typescript

import { useScene } from '@pascal-app/core'

interface AhuPanelProps {
  nodeId: string
```

- **定格値**: coolingCapacity, heatingCapacity, ratedAirflow
- **接続状態**: 各ポートの接続先ダクト/パイプ表示

### 3. 🔵 DiffuserPanel — 制気口プロパティパネル

`packages/editor/src/components/panels/hvac/diffuser-panel.tsx` に制気口プロパティパネルを作成する。

```typescript
// packages/editor/src/components/panels/hvac/diffuser-panel.tsx
import { useScene } from '@pascal-app/core'

interface DiffuserPanelProps {
  nodeId: string
}

export function DiffuserPanel({ nodeId }: DiffuserPanelProps) {
  const node = useScene(state => state.nodes[nodeId])
  if (!node || node.type !== 'diffuser') return null

  return (
    <div>
      <h3>制気口プロパティ</h3>
      {/* tag */}
      {/* subType */}
      {/* neckDiameter */}
      {/* airflowRate */}
      {/* 接続先ダクト */}
    </div>
  )
}
```

表示項目:
- **tag**: 制気口タグ（編集可能、例: SA-01）
- **subType**: 給気/還気/排気の表示
- **neckDiameter**: ネック径（mm）
- **airflowRate**: 割当風量（m3/h）
- **接続先ダクト**: connectedDuctIdの表示（未接続時は「未接続」）

## 単体テスト要件

### テスト1: SystemPanel — 基本表示

- **Given**: SystemNode（name='AHU系統-1', servedZoneIds: 3ゾーン）

- **When**: SystemPanelをレンダリング
- **Then**: 「冷房: 45kW」「暖房: 36kW」「風量: 4500m3/h」が表示される

### テスト3: AhuPanel — 基本表示

### テスト4: AhuPanel — 定格値表示
- **信頼性**: 🔵

- **Then**: 「冷房: 80kW」「暖房: 70kW」「風量: 15000m3/h」が表示される

### テスト5: DiffuserPanel — 基本表示
- **信頼性**: 🔵
