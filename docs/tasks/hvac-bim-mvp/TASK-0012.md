### 2. 🔵 boundary頂点からポリゴンメッシュ生成

HvacZoneNodeのboundary配列（{x, y}の頂点リスト）からThree.jsのShapeGeometryを生成する。

```typescript
function createZoneShape(boundary: { x: number; y: number }[]): Shape {
  const shape = new Shape()
  if (boundary.length < 3) return shape

  shape.moveTo(boundary[0].x, boundary[0].y)
  for (let i = 1; i < boundary.length; i++) {
    shape.lineTo(boundary[i].x, boundary[i].y)
  }
  shape.closePath()
  return shape
}
```

- 最低3頂点が必要
- 頂点が不足する場合は空のShapeを返す
- メッシュはXZ平面上に配置（Y=floorHeight）

```typescript
const ZONE_USAGE_COLORS: Record<string, string> = {
  office_general: '#42A5F5',  // 一般オフィス: 青
  conference: '#FFA726',       // 会議室: オレンジ

```

- opacity: 0.4（半透明）
- DoubleSide描画で裏面からも視認可能
- 選択時はopacity: 0.6に増加

イベントバスへのイベント発行。既存パターンに完全準拠。

- レンダラーはイベントの発行のみ、リスナー登録は行わない

### 6. 🟡 calcResult未設定時はグレー表示

負荷計算結果（calcResult）が未設定のゾーンはデフォルトカラー（グレー）で表示する。

```typescript
const color = useMemo(() => {
  if (!node.calcResult) return ZONE_DEFAULT_COLOR
  return ZONE_USAGE_COLORS[node.usage] ?? ZONE_DEFAULT_COLOR
}, [node.calcResult, node.usage])
```

- calcResult未設定 → グレー(#9E9E9E)
- calcResult設定済み → 用途別カラー
- TASK-0018でフェードインアニメーションを追加予定

## 注意事項

- **ビューア分離**: このレンダラーはpackages/viewerに配置する。editorからのimportは厳禁。phase/mode/useEditorへの参照不可
- **ZONE_LAYER**: 必ず定数 `ZONE_LAYER = 2` を使用し、ハードコーディング禁止
- **パフォーマンス**: 大量のゾーン表示時、ShapeGeometryの再生成を最小化するためuseMemoを活用
- **既存レンダラーとの整合**: Wall/Slab等の既存レンダラーと同じパターン（useRegistry + useNodeEvents）を厳守
