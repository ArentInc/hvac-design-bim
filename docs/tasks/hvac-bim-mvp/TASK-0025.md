### 4. 🔵 風量均等配分 (REQ-604)

ゾーンの必要風量をゾーン内の制気口数で均等に配分する。

```typescript
const redistributeAirflow = (zoneId: string) => {
  const { nodes, updateNode } = useScene.getState()
  const zone = nodes[zoneId]
  if (!zone || zone.type !== 'hvacZone') return

  const requiredAirflow = zone.calcResult?.requiredAirflow ?? 0

  // ゾーン内の制気口を取得
  const diffuserIds = zone.children?.filter(childId => {
    const child = nodes[childId]
    return child?.type === 'diffuser' && child.subType === 'supply'
  }) ?? []

  if (diffuserIds.length === 0) return

  const airflowPerDiffuser = requiredAirflow / diffuserIds.length

  for (const diffuserId of diffuserIds) {
    updateNode(diffuserId, { airflowRate: airflowPerDiffuser })
  }
}
```

- zone.requiredAirflow / diffuserCount で均等配分
- 給気口（supply）のみ対象（還気口・排気口は別計算）
- 制気口の追加・削除時にも再配分が自動実行

## 注意事項

- **Toolsルール遵守**: Three.js APIを直接呼ばない。viewerからインポートしない
- **風量配分の精度**: 割り切れない場合の端数処理（Math.roundまたは最後の制気口に残りを割り当て）
- **maxAirflow超過警告**: 均等配分後に各制気口のmaxAirflowを超過する場合、ユーザーに警告（制気口の追加を推奨）
- **undo/redo**: 配置+風量再配分を1つのundoステップにまとめる検討が必要
