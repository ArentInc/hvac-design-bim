## 実装詳細

### 1. 🔵 DiffuserRendererコンポーネント作成

`packages/viewer/src/components/renderers/hvac/diffuser-renderer.tsx` にレンダラーコンポーネントを作成する。

```typescript
// packages/viewer/src/components/renderers/hvac/diffuser-renderer.tsx
import { useRef } from 'react'
import type { Group } from 'three'
import { useRegistry } from '../../../hooks/use-registry'
import { useNodeEvents } from '../../../hooks/use-node-events'

interface DiffuserRendererProps {
  nodeId: string
  position: [number, number, number]
  neckDiameter: number       // mm
  subType: 'supply' | 'return' | 'exhaust'
}

export function DiffuserRenderer({ nodeId, position, neckDiameter, subType }: DiffuserRendererProps) {
  const ref = useRef<Group>(null)
  useRegistry(nodeId, 'diffuser', ref)
  useNodeEvents('diffuser', nodeId)

  return (
    <group ref={ref} position={position}>
      {/* フェースプレート + ネック */}
    </group>
  )
}
```

- viewerパッケージに配置（レンダラーはviewerの責務）
- editorやapps/editorからインポートしない（Viewer Isolationルール）

### 2. 🔵 正方形フェースプレートメッシュ (REQ-1504)

制気口のフェースプレートを正方形メッシュで表現する。

```tsx
// フェースプレートサイズはネック径の約1.5倍（一般的な制気口の比率）
const faceSize = (neckDiameter / 1000) * 1.5  // mm → m 変換

<mesh rotation={[-Math.PI / 2, 0, 0]}>
  <planeGeometry args={[faceSize, faceSize]} />
  <meshStandardMaterial
    color={subType === 'supply' ? '#42A5F5' : '#EF5350'}
    side={DoubleSide}
  />
</mesh>
```

- 給気口（supply）: 青系カラー
- 還気口（return）: 赤系カラー
- 排気口（exhaust）: グレー系カラー
- 天井面に平行に配置（rotation調整）

### 3. 🔵 ネック円筒メッシュ

フェースプレートの上方にネック円筒を表示する。

```tsx
const neckRadius = neckDiameter / 2 / 1000  // mm → m
const neckHeight = 0.15  // 150mm

<mesh position={[0, neckHeight / 2, 0]}>
  <cylinderGeometry args={[neckRadius, neckRadius, neckHeight, 16]} />
  <meshStandardMaterial color="#90A4AE" />
</mesh>
```

- neckDiameterからネック径を算出（mm→m変換）
- ネック高さは一般的な制気口サイズ（150mm程度）
- フェースプレートの中心から上方に配置

### 4. 🔵 useRegistry登録

sceneRegistryにディフューザーメッシュを登録する。

```typescript
useRegistry(nodeId, 'diffuser', ref)
```

- sceneRegistry.nodesとsceneRegistry.byType.diffuserに登録
- アンマウント時に自動クリーンアップ

### 5. 🔵 useNodeEventsイベント発火

イベントバスにディフューザー関連イベントを発火する。

```typescript
useNodeEvents('diffuser', nodeId)
```

- `diffuser:click`, `diffuser:enter`, `diffuser:leave` 等のイベントを発火
- レンダラーはイベント発火のみ（イベントバスルール）
