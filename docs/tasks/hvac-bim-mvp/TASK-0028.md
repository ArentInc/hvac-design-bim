## 実装詳細

### 1. 🔵 DuctSegmentRendererコンポーネント作成

**ファイル**: `packages/viewer/src/components/renderers/hvac/duct-segment-renderer.tsx`

DuctSegmentNodeのデータを受け取り、start→end間に矩形断面の直線メッシュを描画するReactコンポーネント。

- props: `{ nodeId: string }`
- useScene.getState().nodes[nodeId] からDuctSegmentNodeを取得
- start/end座標からメッシュの位置・回転を計算

### 2. 🔵 start→end間の矩形/丸断面メッシュ生成 (REQ-1502)

DuctSegmentNodeのstart, end座標を用いて直線メッシュを生成する。

- 矩形断面: BoxGeometry(width, height, length) をstart→end方向に回転配置
- 丸断面（将来対応用）: CylinderGeometry(radius, radius, length) — 現MVPでは矩形のみ
- length = start→end間のユークリッド距離
- メッシュの中点を (start + end) / 2 に配置
- lookAt的な回転で方向を設定

### 3. 🔵 寸法未確定時の破線表示

DuctSegmentNodeのwidth/heightが0または未設定の場合、細い破線で表示する。

- 寸法未確定判定: `width === 0 || height === 0`
- 未確定時: LineDashedMaterial + Line で細い点線表示
- 確定時: MeshStandardMaterial + Mesh で断面付きメッシュ表示
- 表示切替はuseEffectでwidht/height変更を監視

### 4. 🔵 useRegistry(nodeId, 'duct_segment', ref)

sceneRegistryへのメッシュ登録。

- `useRegistry(nodeId, 'duct_segment', ref)` を呼び出し
- refはGroup（メッシュ+ラベルを含むコンテナ）
- useLayoutEffectで同期的に登録

### 5. 🔵 useNodeEvents('duct_segment', nodeId)

イベントバスへのイベント発火設定。

- `useNodeEvents('duct_segment', nodeId)` を呼び出し
- click, enter, leave イベントを発火
- レンダラーはイベントをemitのみ、listenしない
