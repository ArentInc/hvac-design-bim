## 実装詳細

### 1. 🔵 packages/editor/src/components/ui/sidebar/panels/zone-list-panel.tsx

左サイドバーに表示するゾーン一覧パネル。

```typescript
import { useScene } from '@pascal-app/core'
import { useViewer } from '@pascal-app/viewer'
import type { HvacZoneNode } from '@pascal-app/core/schema/nodes/hvac-zone'

export function ZoneListPanel() {
  const nodes = useScene((s) => s.nodes)
  const selectedIds = useViewer((s) => s.selectedIds)
  const setSelectedIds = useViewer((s) => s.setSelectedIds)

  // hvac_zoneタイプのノードを抽出
  const zones = useMemo(() => {
    return Object.values(nodes).filter(
      (node): node is HvacZoneNode => node.type === 'hvac_zone'
    )
  }, [nodes])

  return (
    <div className="zone-list-panel">
      <h3>ゾーン一覧</h3>
      {zones.length === 0 ? (
        <p>ゾーンがありません</p>
      ) : (
        <ul>
          {zones.map((zone) => (
            <ZoneListItem
              key={zone.id}
              zone={zone}
              isSelected={selectedIds.includes(zone.id)}
              onSelect={() => setSelectedIds([zone.id])}
            />
          ))}
        </ul>
      )}
    </div>
  )
}
```

### 2. 🔵 全HvacZone一覧表示（名前、用途、面積、負荷結果サマリー）

各ゾーンの基本情報と負荷結果サマリーを行形式で表示する。

```typescript
interface ZoneListItemProps {
  zone: HvacZoneNode
  isSelected: boolean
  onSelect: () => void
}

function ZoneListItem({ zone, isSelected, onSelect }: ZoneListItemProps) {
  const usageLabel = USAGE_LABELS[zone.usage] ?? zone.usage

  return (
    <li
      className={`zone-list-item ${isSelected ? 'selected' : ''}`}
      onClick={onSelect}
    >
      <div className="zone-list-item-header">
        <UsageColorIcon usage={zone.usage} />
        <span className="zone-name">{zone.zoneName || '名称未設定'}</span>
      </div>
      <div className="zone-list-item-details">
        <span className="zone-usage">{usageLabel}</span>
        <span className="zone-area">{zone.floorArea.toFixed(1)} m²</span>
      </div>
      {zone.calcResult && (
        <div className="zone-list-item-load">
          <span>冷房: {formatLoadCompact(zone.calcResult.coolingLoad)}</span>
          <span>風量: {zone.calcResult.requiredAirflow.toLocaleString()} m³/h</span>
        </div>
      )}
    </li>
  )
}

const USAGE_LABELS: Record<string, string> = {
  office_general: '一般オフィス',
  conference: '会議室',
  reception: '受付/ロビー',
  office_server: 'サーバー室',
  corridor: '廊下',
}

function formatLoadCompact(watts: number): string {
  if (watts >= 1000) return `${(watts / 1000).toFixed(1)} kW`
  return `${Math.round(watts)} W`
}
```

- ゾーン名未設定時は「名称未設定」を表示
- calcResult未設定時は負荷サマリー行を非表示
- 面積は小数点1桁表示

### 3. 🔵 クリックで該当ゾーン選択（useViewer.selectedIds更新）

ゾーン一覧のクリックでビューア内のノード選択と連動する。

```typescript
const handleSelect = useCallback(
  (zoneId: string, event: React.MouseEvent) => {
    if (event.ctrlKey || event.metaKey) {
      // Ctrl/Cmd + クリック: マルチ選択
      setSelectedIds((prev) =>
        prev.includes(zoneId)
          ? prev.filter((id) => id !== zoneId)
          : [...prev, zoneId]
      )
    } else {
      // 通常クリック: 単一選択
      setSelectedIds([zoneId])
    }
  },
  [setSelectedIds]
)
```

- 通常クリック: 単一選択（他の選択を解除）
- Ctrl/Meta + クリック: マルチ選択（トグル）
- 選択状態はuseViewer.selectedIdsで管理（ビューア側のハイライトと同期）

### 4. 🟡 用途別カラーアイコン表示

ゾーンの用途に対応するカラーアイコンを表示する。

```typescript
const ZONE_USAGE_COLORS: Record<string, string> = {
  office_general: '#42A5F5',
  conference: '#FFA726',
  reception: '#66BB6A',
  office_server: '#EF5350',
  corridor: '#BDBDBD',
}

function UsageColorIcon({ usage }: { usage: string }) {
  const color = ZONE_USAGE_COLORS[usage] ?? '#9E9E9E'

  return (
    <span
      className="usage-color-icon"
      style={{
        display: 'inline-block',
        width: 12,
        height: 12,
        borderRadius: '50%',
        backgroundColor: color,
        marginRight: 8,
      }}
    />
  )
}
```

- TASK-0012のZONE_USAGE_COLORSと同じカラーマッピング
- 円形アイコン（12px）で表示
- カラー定数は共通モジュールから参照（重複定義回避）
