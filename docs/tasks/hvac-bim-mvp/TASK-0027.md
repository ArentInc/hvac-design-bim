## タスク概要

左パネルに系統ツリー（SystemTreePanel）と機器カタログ（EquipmentCatalogPanel）を表示する。系統ツリーは系統→ゾーン→機器の階層構造をツリービューで表示し、ノードクリックでビューア内の対応オブジェクトを選択する。機器カタログはAHU/制気口のカタログ一覧と選定状態を表示する。

## 実装詳細

### 1. 🔵 SystemTreePanel — 系統ツリー表示

`packages/editor/src/components/panels/hvac/system-tree-panel.tsx` に系統ツリーパネルを作成する。

```typescript
// packages/editor/src/components/panels/hvac/system-tree-panel.tsx
import { useScene } from '@pascal-app/core'

export function SystemTreePanel() {
  const nodes = useScene(state => state.nodes)

  // ノードからツリー構造を構築
  const systemNodes = Object.values(nodes).filter(n => n.type === 'system')

  return (
    <div>
      <h3>系統ツリー</h3>
      {systemNodes.map(system => (
        <SystemTreeItem key={system.id} systemNode={system} />
      ))}
    </div>
  )
}
```

ツリー構造:
```
📁 AHU系統-1
  ├── 🏠 ゾーンA（冷房: 15kW, 風量: 1500m3/h）
  │   ├── 💨 SA-01（300mm, 750m3/h）
  │   └── 💨 SA-02（300mm, 750m3/h）
  ├── 🏠 ゾーンB（冷房: 20kW, 風量: 2000m3/h）
  │   └── 💨 SA-03（400mm, 2000m3/h）
  └── 🔧 AHU-01（型式A-80, 80kW）
📁 AHU系統-2
  └── ...
```

- 系統（SystemNode）を最上位に表示
- 各系統の下にservedZoneIdsのゾーンを展開
- 各ゾーンの下に子ノード（制気口等）を表示
- 系統に紐付くAHUも表示

### テスト6: ノード選択連動 — ツリー→ビューア
