### 2. 🔵 グリッドpointerdownイベントで頂点収集

イベントバスの`grid:pointerdown`を購読して頂点を収集する。

```typescript
useEffect(() => {
  const handler = (e: GridPointerEvent) => {
    // スナップ済み座標を取得
    const point = { x: e.point.x, y: e.point.z } // XZ平面→XY変換
    setVertices((prev) => [...prev, point])

```

### 3. 🔵 Shoelace formulaでリアルタイム面積算出 (REQ-1601)

ポリゴンの面積をShoelace formula（ガウスの面積公式）で算出する。

```typescript
function calculatePolygonArea(vertices: { x: number; y: number }[]): number {
  if (vertices.length < 3) return 0

  let area = 0
  const n = vertices.length
  for (let i = 0; i < n; i++) {
    const j = (i + 1) % n
    area += vertices[i].x * vertices[j].y
    area -= vertices[j].x * vertices[i].y
  }
  return Math.abs(area) / 2
}
```

- 頂点追加のたびにリアルタイム再計算
- currentPoint（マウス位置）を含む仮のポリゴンでプレビュー面積を表示
- 単位: m²

- 用途はデフォルトで'office_general'、後からプロパティパネルで変更可能

### 6. 🟡 最小面積チェック (EDGE-001: 面積0以下は拒否)

面積が0以下のポリゴンを拒否する。

```typescript
if (floorArea <= 0) {
```

## 注意事項

- **ツール分離**: ツールはpackages/editorに配置。Three.js APIを直接呼び出さないこと。プレビュー描画はJSX（R3F）で行う
- **イベントバスルール**: リスナー登録と解除は同じ関数参照を使用すること。useEffect内でon/offを対にする
- **プレビュー管理**: プレビュージオメトリはツールのローカル状態。シーンストアに保存しない
- **座標変換**: グリッドイベントの座標（XZ平面）からHvacZoneNodeのboundary座標（XY平面）への変換に注意
