## 実装詳細

### 1. 🔵 ゾーン用途別カラー最終確認 (REQ-1501)

HvacZoneNodeの用途（usage）に応じたフロア塗りつぶしカラーを確認・調整する。

- 事務室: 青系（例: #42A5F5）
- 会議室: 緑系（例: #66BB6A）
- サーバー室: 赤系（例: #EF5350）
- 廊下/共用部: グレー系（例: #BDBDBD）
- 各カラーのアルファ値をZONE_LAYER透過表示に最適化
- カラー定数をpackages/core/src/constants/に集約管理

### 2. 🔵 ダクト接続状態別色・太さ最終調整 (REQ-1502)

DuctSegmentNodeの接続状態・寸法確定状態に応じた表示を確認・調整する。

- 接続済み+寸法確定: ソリッド表示、設計寸法の断面メッシュ
- 接続済み+寸法未確定: 細い破線表示（グレー）
- 未接続: 赤い点線表示
- 選択時のハイライト色との組み合わせ確認
- DuctVisualSystem（TASK-0033）で実装済みの表示ロジックの微調整

### 3. 🔵 配管色分け最終確認 (REQ-1503)

PipeSegmentNodeの冷水/温水の色分けを確認・調整する。

- 冷水配管: #0288D1（ライトブルー）
- 温水配管: #01579B（ダークブルー）
- 冷温水兼用: グラデーションまたはストライプ表示（MVPではダークブルー統一も可）
- 配管太さ: diameterに応じたスケーリング
- PipeVisualSystem（TASK-0037）で実装済みの表示ロジックの微調整

### 4. 🔵 右パネル表示フォーマットPRDセクション21.5準拠 (REQ-1505)

選択ノードの右パネル（PropertyPanel）での表示フォーマットを統一する。

- ゾーン選択時: 用途、面積、冷房負荷(W)、暖房負荷(W)、所属系統
- 機器選択時: 型番、能力(kW)、風量(m³/h)、接続ポート状態
- ダクト選択時: 寸法(W×H mm)、風量(m³/h)、風速(m/s)、圧損(Pa)
- 配管選択時: 口径(mm A)、流量(L/min)、接続先
- 制気口選択時: 型番、風量(m³/h)、設定風量
- 数値の単位表示と小数点桁数の統一
