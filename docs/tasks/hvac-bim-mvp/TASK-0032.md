## 実装詳細

### 1. 🔵 DuctSizingSystemコンポーネント作成

**ファイル**: `packages/core/src/systems/hvac/duct-sizing-system.tsx`

Coreシステムとして実装。AirflowDistributionSystem完了後にdirtyNodesを監視し、風量が設定されたDuctSegmentに対して寸法を自動算出する。

- dirtyNodesにDuctSegmentが含まれ、airflowRateが設定済みの場合にトリガー
- AirflowDistributionSystemの後に実行される必要がある（依存順序）

### 2. 🔵 等速法: 断面積 = airflowRate / targetVelocity (REQ-901)

等速法に基づく断面積計算。

- **計算式**: `A = Q / v`
  - A: 断面積 (m²)
  - Q: 風量 (m³/s) — DuctSegmentのairflowRate(m³/h)を3600で除算
  - v: 目標風速 (m/s)
- 断面積から矩形寸法（width × height）を算出
- 正方形に近い比率を優先（アスペクト比最小化）

### 3. 🔵 幹線推奨風速6~8 m/s、枝線3~5 m/s (REQ-902)

ダクト区間の種別に応じた推奨風速の適用。

- **幹線判定**: AHU直近セグメントまたはT分岐前のセグメント → 風速7 m/s（中央値）
- **枝線判定**: T分岐後の末端セグメント → 風速4 m/s（中央値）
- 判定ロジック: グラフ上でのAHUからの距離、または下流Diffuser数で分類
- ユーザーが手動で targetVelocity を設定した場合はそれを優先

### 4. 🔵 standard-duct-sizes.jsonスナップ (REQ-902)

計算された寸法を標準サイズにスナップする。

- `packages/core/src/data/standard-duct-sizes.json` から標準寸法リスト読み込み
- 標準寸法例: 150, 200, 250, 300, 350, 400, 450, 500, 600, 750, 900, 1000 (mm)
- 計算値以上の最小標準サイズにスナップ
- width, height をそれぞれ独立にスナップ

### 5. 🔵 アスペクト比≤4.0制約 (REQ-903)

矩形ダクトのアスペクト比を制約する。

- アスペクト比 = max(width, height) / min(width, height)
- 4.0を超える場合、短辺を拡大して4.0以下に調整
- 調整後も標準サイズにスナップ
- 制約違反時の警告ログ出力

### 6. 🔵 updateNode(ductId, {width, height})

計算結果をDuctSegmentNodeに反映する。

- `updateNode(ductId, { width, height })` で寸法を更新
- 更新後、DuctVisualSystem（TASK-0033）が太さ・色を再描画
- 変更があったノードのみ更新

## 単体テスト要件

**テストファイル**: `packages/core/src/systems/hvac/__tests__/duct-sizing-system.test.ts`

### テスト1: 基本的な等速法計算
- **信頼性**: 🔵
- **Given**: airflowRate = 3600 m³/h (= 1.0 m³/s), targetVelocity = 5.0 m/s
- **When**: 寸法選定を実行
- **Then**: 断面積 = 0.2 m², 約 450mm × 450mm (標準サイズスナップ後)

### テスト2: 幹線風速適用
- **信頼性**: 🔵
- **Given**: AHU直近のDuctSegment, airflowRate = 7200 m³/h
- **When**: 寸法選定を実行
- **Then**: 幹線風速7 m/sが適用され、断面積 ≈ 0.286 m²

### テスト3: 枝線風速適用
- **信頼性**: 🔵
- **Given**: 末端のDuctSegment, airflowRate = 360 m³/h
- **When**: 寸法選定を実行
- **Then**: 枝線風速4 m/sが適用され、断面積 ≈ 0.025 m²

### テスト4: 標準サイズスナップ
- **信頼性**: 🔵
- **Given**: 計算結果 width=273mm, height=273mm
- **When**: 標準サイズスナップ
- **Then**: width=300mm, height=300mm にスナップされる

### テスト5: アスペクト比制約
- **信頼性**: 🔵
- **Given**: 計算結果 width=1000mm, height=200mm（アスペクト比5.0）
- **When**: アスペクト比制約を適用
- **Then**: height≥250mm に調整（アスペクト比4.0以下）

### テスト6: アスペクト比制約後の標準サイズスナップ
- **信頼性**: 🔵
- **Given**: アスペクト比制約で height=240mm に調整
- **When**: 標準サイズスナップ
- **Then**: height=250mm にスナップされる

### テスト7: 風量ゼロ時のスキップ
- **信頼性**: 🔵
- **Given**: airflowRate = 0 のDuctSegment
- **When**: 寸法選定を実行
- **Then**: 寸法選定がスキップされ、width/height は変更されない
