## 実装詳細

### 1. 🔵 PipeSegmentRendererコンポーネント作成

**ファイル**: `packages/viewer/src/components/renderers/hvac/pipe-segment-renderer.tsx`

PipeSegmentNodeのデータを受け取り、start→end間に円筒メッシュを描画するReactコンポーネント。

- props: `{ nodeId: string }`
- useScene.getState().nodes[nodeId] からPipeSegmentNodeを取得
- start/end座標からメッシュの位置・回転を計算

### 2. 🔵 媒体種別による色分け (REQ-1503)

PipeSegmentNodeのmediumに応じたマテリアルカラー設定。

- **chilled_water_supply（冷水往管）**: #0288D1（明るい青）
- **chilled_water_return（冷水還管）**: #01579B（暗い青）
- **hot_water_supply（温水往管）**: #E53935（赤） — 将来対応
- **hot_water_return（温水還管）**: #B71C1C（暗い赤） — 将来対応
- MeshStandardMaterial でマテリアル生成

### 3. 🔵 口径比例の太さ表示

PipeSegmentNodeのdiameterに基づいて円筒の太さを設定する。

- CylinderGeometry(radius, radius, length, segments) で円筒生成
- radius = diameter / 2（実寸法）
- 最小描画半径を設定（口径未設定時の見やすさ確保）
- length = start→end距離
- segments = 8（パフォーマンスとのバランス）

### 4. 🔵 useRegistry + useNodeEvents

sceneRegistryへの登録とイベント発火。

- `useRegistry(nodeId, 'pipe_segment', ref)` — Groupをrefとして登録
- `useNodeEvents('pipe_segment', nodeId)` — click, enter, leave イベント
- 既存レンダラーパターンに準拠

## 注意事項

- **Viewer Isolation**: packages/viewerはpackages/editorに依存してはならない
- **配管色定数**: カラーコードは定数ファイルに外部定義し、DuctVisualSystemのカラー定数と統一管理
- **口径表示**: 実寸法で描画すると配管が細すぎて見えない場合がある。最小描画半径のしきい値設定が必要
- **パフォーマンス**: CylinderGeometryのsegments数はパフォーマンスに影響。8セグメントが推奨バランス
