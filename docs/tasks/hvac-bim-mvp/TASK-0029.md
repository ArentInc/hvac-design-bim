## 実装詳細

### 1. 🔵 DuctFittingRendererコンポーネント作成

**ファイル**: `packages/viewer/src/components/renderers/hvac/duct-fitting-renderer.tsx`

DuctFittingNodeのデータを受け取り、継手タイプに応じた3Dメッシュを描画するReactコンポーネント。

- props: `{ nodeId: string }`
- useScene.getState().nodes[nodeId] からDuctFittingNodeを取得
- fittingType（elbow_90, tee, cross等）に応じてメッシュ生成を分岐

### 2. 🟡 エルボ90°: 曲がりメッシュ生成

エルボ90°継手用の曲がりメッシュを生成する。

- TorusGeometry(radius, tubeRadius, radialSegments, tubularSegments, arc) で90°アーク生成
- radius: ダクト断面幅に基づく曲げ半径
- arc: Math.PI / 2（90度）
- ポート2点の方向に合わせて回転配置
- 簡易版: 2つの短いBoxGeometryを45°で接続する代替手法も可

### 3. 🟡 T分岐: 3方向分岐メッシュ

T分岐継手用の3方向分岐メッシュを生成する。

- 主管: 直線BoxGeometry
- 分岐管: 主管中央から垂直方向にBoxGeometry
- 接合部: 交差部分の結合メッシュ
- 3つのポート位置に合わせて配置

### 4. 🔵 ポート位置の円筒表示

各ポートの接続位置を小さな円筒メッシュで視覚化する。

- CylinderGeometry(portRadius, portRadius, portDepth) で円筒生成
- ポートごとにposition座標に配置
- direction方向に向きを設定
- 接続済み: 緑色、未接続: 赤色で色分け

### 5. 🔵 useRegistry + useNodeEvents

sceneRegistryへの登録とイベント発火。

- `useRegistry(nodeId, 'duct_fitting', ref)` — Groupをrefとして登録
- `useNodeEvents('duct_fitting', nodeId)` — click, enter, leave イベント
- 既存レンダラーパターンに準拠
