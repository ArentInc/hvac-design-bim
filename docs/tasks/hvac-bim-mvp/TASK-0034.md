## 実装詳細

### 1. 🔵 PressureLossSystemコンポーネント作成

**ファイル**: `packages/core/src/systems/hvac/pressure-loss-system.tsx`

Coreシステムとして実装。DuctSizingSystem完了後にdirtyNodesを監視し、寸法が確定したDuctSegmentの圧損を計算する。

- dirtyNodesにDuctSegmentが含まれ、width/heightが確定済みの場合にトリガー
- systemId単位で全経路の圧損を再計算

### 2. 🔵 直管圧損: ダルシー・ワイスバッハ式ベース (REQ-1001)

ダクト直管部の圧力損失をダルシー・ワイスバッハ式で計算する。

- **計算式**: `ΔP = λ × (L / Dh) × (ρ × v² / 2)`
  - λ: 摩擦係数（galvanized: 0.02、一般的な概算値）
  - L: ダクト長さ (m) — start→end距離
  - Dh: 等価直径 (m) — `Dh = 2 × width × height / (width + height)` （矩形ダクト）
  - ρ: 空気密度 (kg/m³) — 1.2（標準状態）
  - v: 風速 (m/s) — `airflowRate(m³/s) / (width × height)`
- 結果は Pa 単位

### 3. 🔵 flexible材質: 摩擦係数×4 (REQ-1003)

ダクト材質がflexibleの場合、摩擦係数を4倍にする。

- DuctSegmentNode.material === 'flexible' の場合: λ = 0.02 × 4 = 0.08
- DuctSegmentNode.material === 'galvanized'（デフォルト）の場合: λ = 0.02
- 材質判定はDuctSegmentNodeのmaterialフィールドで行う

### 4. 🔵 継手損失: エルボ90°=0.3, T分岐=0.5 等 (REQ-1002)

ダクト継手の局所損失を計算する。

- **局所損失計算式**: `ΔP_local = ζ × (ρ × v² / 2)`
  - ζ: 局所損失係数
  - v: 継手通過風速 (m/s)
- **局所損失係数テーブル**:
  - エルボ90°: ζ = 0.3
  - T分岐（直進）: ζ = 0.5
  - T分岐（分岐側）: ζ = 1.0
  - レデューサー: ζ = 0.2
- 各DuctFittingNodeの fittingType に応じてζを選択

### 5. 🔵 最遠経路の総圧損算出 (REQ-1004)

AHUから最も圧損が大きい経路（最遠経路）の総圧損を算出する。

- AHUをルートとしてグラフの全経路を探索
- 各経路: 直管圧損の合計 + 経路上の継手損失の合計
- 全経路の中で最大の総圧損を「最遠経路圧損」として選択
- ダイクストラ法的なアプローチ、または全経路列挙（ダクト系統は小規模のためDFS十分）

### 6. 🔵 requiredFanPressure計算

最遠経路圧損からAHU必要送風静圧を算出する。

- `requiredFanPressure = 最遠経路総圧損 × 安全率（1.1）`
- AHUの staticPressure と比較して余裕度を算出
- 余裕度不足（requiredFanPressure > staticPressure）の場合に警告

### 7. 🔵 updateNode(systemId, {requiredFanPressure})

計算結果をHvacSystemNodeに反映する。

- `updateNode(systemId, { requiredFanPressure })` で送風圧反映
- 各DuctSegmentにも個別の `pressureLoss` を更新
- 更新後dirtyNodesクリア

## 単体テスト要件

**テストファイル**: `packages/core/src/systems/hvac/__tests__/pressure-loss-system.test.ts`

### テスト1: 基本的な直管圧損計算
- **信頼性**: 🔵
- **Given**: L=10m, width=0.4m, height=0.3m, airflowRate=3600m³/h, material='galvanized'
- **When**: 直管圧損計算
- **Then**: ΔP が物理的に妥当な値（数Pa～数十Pa）で算出される

### テスト2: flexible材質の摩擦係数4倍
- **信頼性**: 🔵
- **Given**: 同一条件でmaterial='galvanized'とmaterial='flexible'
- **When**: それぞれ直管圧損計算
- **Then**: flexible の圧損が galvanized の約4倍

### テスト3: エルボ90°の継手損失
- **信頼性**: 🔵
- **Given**: エルボ90°継手、通過風速5 m/s
- **When**: 継手損失計算
- **Then**: ΔP = 0.3 × (1.2 × 25 / 2) = 4.5 Pa

### テスト4: T分岐の継手損失
- **信頼性**: 🔵
- **Given**: T分岐継手（分岐側）、通過風速5 m/s
- **When**: 継手損失計算
- **Then**: ΔP = 1.0 × (1.2 × 25 / 2) = 15.0 Pa

### テスト5: 最遠経路の選択
- **信頼性**: 🔵
- **Given**: AHU→分岐→経路A(30Pa), 経路B(50Pa)
- **When**: 最遠経路算出
- **Then**: 経路B(50Pa)が最遠経路として選択される

### テスト6: requiredFanPressure計算
- **信頼性**: 🔵
- **Given**: 最遠経路総圧損 = 100 Pa
- **When**: requiredFanPressure算出
- **Then**: requiredFanPressure = 110 Pa（安全率1.1）

### テスト7: 等価直径計算（矩形ダクト）
- **信頼性**: 🔵
- **Given**: width=0.4m, height=0.3m
- **When**: 等価直径算出
- **Then**: Dh = 2 × 0.4 × 0.3 / (0.4 + 0.3) ≈ 0.343m
